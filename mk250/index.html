Prepare documents for Uganda Immigration

This tool is specifically designed to prepare documents for uploading to Uganda Electronic Visa/Permit Application System which requires any uploaded image file to have a file size not exceeding 250kb. This tool resizes and compresses source files to fit those requirements while maintaining readability (provided the source is readable).

Cropping and scaling are done on your device, no data is uploaded to any server so your documents are secure.

Select document

File must be JPG, JPEG or PNG

Select Document Type

Crop your document

Crop and rotate your document. Move and resize the cropbox on the canvas found at the right of your screen (mobile users: bottom).

Download

The resulting file has a maximum file size of 250kb while maintaining enough detail in the document for readability.
